# Installation Produktivversion 1.0.3

1. Inhalt des ZIP-Archivs entpacken.
2. Im Produktiv-Repository den bisherigen Inhalt ersetzen, `.git` nicht löschen.
3. Prüfen, dass `CNAME` im Hauptverzeichnis liegt und `planner.deepskyastrophoto.de` enthält.
4. Commit und Push auf `main`.
5. Nach dem GitHub-Pages-Deploy `https://planner.deepskyastrophoto.de/VERSION.json` prüfen.

Hinweis: Bestehende Nutzerdaten in IndexedDB bleiben erhalten.
